# Matching-Tool

Matching-Tool für Recruiter - Verbindet Jobs mit passenden Kandidaten.

## Features

- Automatisches Matching von Kandidaten und Jobs
- Integration mit Recruit CRM
- KI-gestützte Analyse mit OpenAI
- Geo-basiertes Matching mit PostGIS

## Setup

```bash
pip install -e .
uvicorn app.main:app --reload
```

## Environment Variables

See `.env.example` for required environment variables.
